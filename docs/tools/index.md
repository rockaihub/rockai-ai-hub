# AI 工具导航总览

本页用于快速了解不同 AI 工具的定位、适合场景和推荐人群。你可以先按任务类型选择工具，再进入具体页面阅读详细指南。

## 工具对比

| 工具名称 | 适合场景 | 推荐人群 | 特点 |
| --- | --- | --- | --- |
| ChatGPT | 写作、问答、办公、学习、编程辅助 | 普通用户、学生、职场人士、开发者 | 通用能力强，生态成熟 |
| Claude | 长文档、写作润色、资料分析 | 研究人员、内容创作者、产品经理 | 长文本体验好，表达自然 |
| Gemini | 搜索增强、多模态、Google 生态 | 学生、办公用户、Google 用户 | 适合资料整理与多模态任务 |
| Grok | 热点、社交平台、短内容 | 内容运营、资讯关注者 | 贴近社交平台语境 |
| Suno | 音乐生成、歌曲 Demo、短视频配乐 | 创作者、音乐爱好者 | 快速生成音乐创意 |
| Cursor | AI 编程、项目理解、重构辅助 | 开发者、编程学习者 | 面向真实代码工作流 |
| Perplexity | 信息检索、带来源问答 | 学生、研究人员、知识工作者 | 适合查资料和追踪来源 |
| Midjourney | 图像生成、视觉创意 | 设计师、创作者、运营人员 | 适合视觉探索和概念图 |

## 按需求选择

### 我是新手

建议先从 [ChatGPT](/chatgpt) 或 [Gemini](/gemini) 开始，先熟悉 AI 对话、提示词和基础工作流。

### 我需要处理文档

优先了解 [Claude](/claude)。如果你经常整理论文、报告、会议纪要和产品文档，它会更贴近文档工作流。

### 我需要写代码

优先了解 [Cursor](/cursor)。它更适合在真实项目中阅读代码、修改文件、排查错误和补充测试。

### 我做内容创作

可以组合使用 ChatGPT、Claude、Suno 和 Midjourney。文字、脚本、音乐、视觉分别选择更适合的工具。

### 我关注热点和社交内容

可以了解 [Grok](/grok)，适合热点观察、观点整理和短内容生成。
